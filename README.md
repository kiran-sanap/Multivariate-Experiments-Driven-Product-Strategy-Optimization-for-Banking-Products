# Multivariate-Experiments-Driven-Product-Strategy-Optimization-for-Banking-Products
Overview

This project applies Business Experiments & Analytics and the Lean Six Sigma (DMADV) framework to redesign Fixed Deposit (FD) banking products based on customer preferences.
The goal was to identify reasons for declining FD satisfaction and validate a data-driven product strategy using real customer data.
